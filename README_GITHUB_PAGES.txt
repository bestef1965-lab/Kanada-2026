Kanada Reiseführer 2026 – GitHub Pages Komplettpaket mit Bildern

Dieses Paket enthält:
- index.html
- 404.html
- assets/ mit Titel- und Kapitelbildern
- PDF-Version
- docs/ Spiegelung für GitHub Pages /docs
- .nojekyll

Empfohlene GitHub-Pages-Einstellung:
1. ZIP entpacken.
2. Den INHALT des entpackten Ordners in dein GitHub-Repository hochladen.
3. GitHub > Repository > Settings > Pages
4. Source: Deploy from a branch
5. Branch: main
6. Folder: /root
7. Save
8. 1-5 Minuten warten.

Alternative:
Wenn /root nicht funktioniert:
- Folder: /docs wählen.

Wichtig:
- Nicht die ZIP hochladen.
- Nicht den äußeren Ordner als Unterordner hochladen.
- index.html muss direkt in der obersten Repository-Ebene sichtbar sein.
